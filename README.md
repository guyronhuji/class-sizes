# Racah Teaching Dashboards

Static dashboards for browsing Racah Institute of Physics lecture enrollment history from 2000 to 2026.

## Pages

- `index.html`: landing page
- `professors.html`: compare selected professors on one combined chart
- `classes.html`: compare selected classes on one combined chart

## Data

- `class-size-data.csv`: source data
- `dashboard-data.js`: precomputed browser-ready data derived from the CSV

## Local use

Open `index.html` in a browser, or publish the repository with GitHub Pages from the repository root.
